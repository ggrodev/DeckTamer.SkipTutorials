SkipTutorial for MelonLoader

SkipTutorials Mod disable introductory fights on the game. 

Dependencies:
MelonLoader Installed on the game: https://melonwiki.xyz/#/README?id=melonloader-installation

How to Build:
Run the game at least once with MelonLoader installed
Set GameFolder on SkipTutorial.csproj
Build SkipTutorial.csproj
